# Ciisa

Desarrollo evaluación 3

Requerimientos
    1. El programa deberá ser capaz de tomar un fichero (archivo con extensión .txt) y leer los datos
       que en élexisten
       
    2. El programa debe ser capaz de presentar un menú de opciones en un Cuadro de Dialogo a través
       de una Lista Desplegable en donde indique las siguientes opciones,según lo indica la imagen
       
    2.1. Ingrese Apellido
    
    2.2. Ingrese RUT (sin DV)
    
    2.3. Salir    
    
    3. Al ingresar una de las opciones anteriores el programa solicitar al usuario que escriba el nombre
       o RUT del alumno (dependiendo de la opción seleccionada) para buscar en la base de datos
       (archivo de texto)
       
    4. El programa deberá indicar el promedio de las notas que están ingresadas del alumno e indicarsi
       está APROBADO o REPROBADO, tal como lo indica la imagen
       
    5. Al ingresar el apellido debe estar validada la opción para solo sea capaz de ingresar textos
    
    6. Al ingresar el RUT debe estar validada la opción para solo sea capaz de ingresar números
    
    7. Al ingresar el RUT se debe agregar automáticamente el DV (digito verificador) para así poder
    buscarlo en la base de datos (archivo de texto)
